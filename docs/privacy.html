Privacy Policy

Last updated: March 4, 2026

This Privacy Policy describes how the "YouTube Live Chat Summarizer" Chrome extension handles user data. This extension is developed and maintained by Kimiya Kitani as a personal project.

1. Information Collection and Use

  • Chat Logs: The extension reads YouTube Live chat messages solely to generate summaries. This data is processed in real-time.
  • Gemini API Key: Your personal API key is required to use the summarization feature. This key is stored locally in your browser's storage.

2. Data Transmission

No data is sent to the developer's server. All chat data is sent directly from your browser to the Google Gemini API. The developer has no access to your chat logs or your API key.

3. Third-Party Services

This extension uses the Google Gemini API. Please refer to Google's Privacy Policy and Terms for information on how they handle data.

プライバシーポリシー

このプライバシーポリシーは、Chrome拡張機能「YouTube Live Chat Summarizer」におけるユーザーデータの取り扱いについて説明するものです。本拡張機能は、日紫喜公一（Kimiya Kitani）個人によって開発・運営されています。

1. データの収集と利用

  • チャットログ: 本拡張機能は、要約を生成する目的のみでYouTubeライブのチャットメッセージを読み取ります。このデータはリアルタイムで処理されます。
  • Gemini APIキー: 要約機能を利用するためにユーザー自身のAPIキーが必要です。このキーはブラウザのローカルストレージにのみ保存されます。

2. データの転送

データが開発者のサーバーに送信されることはありません。 すべてのチャットデータは、お使いのブラウザからGoogle Gemini APIへ直接送信されます。開発者がユーザーのチャット内容やAPIキーにアクセスすることはありません。

3. 第三者サービス

本拡張機能はGoogle Gemini APIを使用しています。Googleによるデータの取り扱いについては、Google Gemini APIの利用規約をご確認ください。
